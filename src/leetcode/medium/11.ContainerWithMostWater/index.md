# 11. Container With Most Water

정수 배열 height가 주어집니다. 배열의 길이는 n이며, n개의 세로선이 x축 위에 그려집니다. i번째 세로선의 두 끝점은 (i, 0)과 (i, height[i])에 위치합니다.

이 중 두 개의 세로선을 선택해 x축과 함께 물을 담을 수 있는 컨테이너를 만들려고 합니다. 컨테이너가 담을 수 있는 최대 물의 양을 구하세요.

_컨테이너 벽은 기울일 수 없습니다._

### 예시 1:

![](https://s3-lc-upload.s3.amazonaws.com/uploads/2018/07/17/question_11.jpg)

입력: height = [1,8,6,2,5,4,8,3,7]  
출력: 49  
설명: 위 그림처럼 [1,8,6,2,5,4,8,3,7]로 그려진 세로선 중 최대 물을 담을 수 있는 법은 49입니다.

### 예시 2:

입력: height = [1,1]  
출력: 1

### 제약사항:

- n == height.length
- 2 <= n <= 10^5
- 0 <= height[i] <= 10^4
