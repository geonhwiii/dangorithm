# 209. Minimum Size Subarray Sum

양의 정수로 이루어진 배열 nums와 양의 정수 target이 주어질 때, 합이 target 이상이 되는 연속 부분 배열(서브배열) 중 길이가 가장 짧은 것을 찾아 그 길이를 반환하세요.

만약 그러한 서브배열이 없다면 0을 반환하세요.

### 예시 1:

입력: target = 7, nums = [2,3,1,2,4,3]
출력: 2
설명: [4,3]이라는 서브배열의 길이가 2로, 조건을 만족하는 최소 길이입니다.

### 예시 2:

입력: target = 4, nums = [1,4,4]
출력: 1

### 예시 3:

입력: target = 11, nums = [1,1,1,1,1,1,1,1]
출력: 0

### 제약 사항:

- 1 <= target <= 109
- 1 <= nums.length <= 105
- 1 <= nums[i] <= 104

심화: O(n) 풀이를 생각해 냈다면, O(n log n) 풀이도 작성해 보세요.
