# 15. 3Sum

정수 배열 nums가 주어졌을 때, 서로 다른 인덱스 i, j, k에 대해 nums[i] + nums[j] + nums[k] == 0을 만족하는 모든 삼중쌍 [nums[i], nums[j], nums[k]]을 구하여라.

단, 결과에 중복된 삼중쌍이 포함되면 안 된다.

### 예시 1:

입력: nums = [-1,0,1,2,-1,-4]
출력: [[-1,-1,2],[-1,0,1]]
설명: 
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0
중복되지 않은 삼중쌍은 [-1,0,1]과 [-1,-1,2]이다.
출력의 순서와 삼중쌍의 순서는 중요하지 않다.

### 예시 2:

입력: nums = [0,1,1]
출력: []
설명: 가능한 삼중쌍이 없으므로 빈 배열을 반환한다.

### 예시 3:

입력: nums = [0,0,0]
출력: [[0,0,0]]
설명: 유일한 삼중쌍이 0이 되므로 [[0,0,0]]을 반환한다.

### 제한 사항:
- 3 <= nums.length <= 3000
- -105 <= nums[i] <= 105
