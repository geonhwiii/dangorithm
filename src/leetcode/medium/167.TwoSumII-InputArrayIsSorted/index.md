# 167. 두 수의 합 II - 입력 배열이 정렬되어 있음

비내림차순(오름차순 및 같은 값 허용)으로 정렬된 1-인덱스 배열 `numbers`가 주어집니다. 이 중에서 두 수를 선택하여 합이 주어진 `target`이 되도록 하세요. 이때 선택된 두 수는 각각 numbers[index1], numbers[index2] (1 <= index1 < index2 <= numbers.length) 입니다.

두 수의 인덱스(index1, index2)를 각각 1부터 시작하는 정수 배열 형태 [index1, index2]로 반환하세요.

모든 테스트 케이스는 정답이 정확히 하나만 존재하도록 생성됩니다. 같은 원소를 두 번 사용할 수 없습니다.

해결 시, 반드시 상수 크기의 추가 공간만 사용할 수 있습니다.

### 예제 1:

입력: numbers = [2,7,11,15], target = 9  
출력: [1,2]  
설명: 2 + 7 = 9이므로 index1=1, index2=2를 반환합니다.

### 예제 2:

입력: numbers = [2,3,4], target = 6  
출력: [1,3]  
설명: 2 + 4 = 6이므로 index1=1, index2=3을 반환합니다.

### 예제 3:

입력: numbers = [-1,0], target = -1  
출력: [1,2]  
설명: -1 + 0 = -1이므로 index1=1, index2=2를 반환합니다.

### 제약 조건:

- 2 <= numbers.length <= 3 \* 10^4
- -1000 <= numbers[i] <= 1000
- numbers는 비내림차순 정렬됨
- -1000 <= target <= 1000
- 모든 입력에 정답은 정확히 하나만 있습니다.
